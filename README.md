\# Brute-Force Detection Dashboard



A comprehensive security monitoring dashboard for detecting and visualizing brute-force attacks through Windows Event ID 4625 monitoring.



!\[Dashboard Preview](assets/images/dashboard-preview.png)



\## Features



\- \*\*Real-time Monitoring\*\*: Track failed login attempts across your network

\- \*\*Interactive Visualizations\*\*: Charts and graphs showing attack patterns

\- \*\*Alert System\*\*: Automatic detection of suspicious activity

\- \*\*Filtering \& Search\*\*: Drill down into specific time ranges, IPs, or users

\- \*\*Export Capabilities\*\*: Generate reports for further analysis



\## Technologies Used



\- HTML5, CSS3, JavaScript

\- Bootstrap 5 for responsive UI

\- Chart.js for data visualization

\- Font Awesome for icons
